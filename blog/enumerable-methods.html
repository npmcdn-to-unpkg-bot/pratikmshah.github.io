Use a Map

(October 31, 2015)

The enumerable class in ruby houses some powerful methods that allow you to iterate through a collection of objects by passing values in succession. One of these methods called map allows you to iterate throught a block and return an array that is identical in size. It is a great method to use with a collection of items if you want to manipulate such as, capitalize all strings in the array.


Let’s do a problem together (figure-1). We have an array from 1 to 6 and we would like to square each number. To setup so we could use the map method we need to write the name of the array and then use the dot operator followed by map (array_name.map). Next, you need to create a code block using brackets ({ }). Then in between the brackets we setup our bars and a variable that will be our element iterator (|x|). Last, we write our statement that we want to compute; in this case it will be the element times itself to get the square (x * x). Note, if you have multiple statements you need to process use the do-end code block.



Ok so what is exactly happening here (figure-2)? Well, the map method is processing each element one by one into a result array that will return after it is done iterating. The return value is the same size as the original array; so if length of array was 6 we will receive a 6 element array. Another, method that does the same exact thing as map is called collect and it is used exactly like map. One, last thing you might say why not use the each iterator instead? The each iterator is different as it returns a single element value one by one and not a full array. Also, it doesn’t return an identical size array as map does.



So, I hope you were able to get a quick grasp about the map method and how useful it will be when you develop. For more information check out the documentation here.
