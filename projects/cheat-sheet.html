Ruby Methods Cheat Sheet

Arrays

any? {|obj| block} - return T/F if obj exists in array

clear - clears all elements in array

map(!) {|element| block} - iterates and returns an array copy

compact(!) - returns copy of array without nil elements

concat(array2) - combines 2nd array into 1st array

count - returns the number of elements

delete(obj) - delete object in array if not found returns nil

delete_at(index) - deletes element at specified index

delete_if{|item| block} - delete element if condition is met

drop(number) - drops the number of elements specified from array

each {|obj| block} - iterates through all elements in array

each_index {|index| block} - same as each but passes index

empty? - return T/F is array is empty

eql?(array2) - returns T/F if array 1 equals array 2

fetch(index) - returns value at index

first(number) - returns first number or number specified

flatten! - converts mu

include?(obj) - returns T/F is object exists in array

insert(index, obj) - inserts data into index specified

join(seperate) - returns string by combining all elements

keep_if {|item| block} - keeps item if condition is met

last(number) - like first but returns last elements

length - returns number of elements in array

push(obj) or << - adds the data at the end of array

replace(array2) - replaces contents of array2 into array1

reverse(!) - reversed order of elements in array

sample(n) - a random element is selected unless speci by n

shuffle(!) - randomize the order of elements in array

sort - sorts array in order

Hashes

any? {|key, value| block} - returns T/F if value exists

clear - clears all values in hash

delete(key) - deletes the key and its value in hash

delete_if {|key, value| block} - iterates and deletes key if true

each_key {|key| block} - iterates by passing key

each_value {|value| block} - iterates by passing value

each_pair {|key, value| block} - iterates and passes key/value

empty? - returns T/F

fetch(key) - returns value of the key

flatten - flattens hash into an array

has_key?(key) - returns T/F if key is in hash

has_value?(value) - returns T/F if value is in hash

include?(key) - same as has key

key(value) - returns the key for the value

keys - returns an array with all they keys in the hash

length - returns number of keys in hash

merge(!)(hash2) - merges hash2 into hash1

replace(hash2) - replaces contents of hash1 with hash2

store(key, value) - adds key-value to hash

values - returns values of hash

values_at(key..) - returns values of the key in an array

Strings

gets.chomp - get user input and cut off new line \n

chomp - will remove \n, \r from end of string

capitalize(!) - returns string with first char capitalized

chars - retunrs an array with each element as one char of string

chop - will remove last char from string

chr - returns first character at beginning of string

delete(!)(string) - deletes string from string

downcase! - convert string chars to lowercase

each_char {|char| block} - iterates through string by each char

end_with?(string) - returns T/F if string ends with letters

include?(str) - returns T/F if piece of the string exists in content

replace(string) - replaces string1 with string2

insert(index, str) - inserts string into specified index

length - returns length of string

reverse(!) - reverses the string

slice(index) or (start, end) - splits the string

split(pattern) - splits the string into an array following the pattern

squeeze(!) - returns string eliminating reoccuring letters and spaces

start_with?(prefix) - returns T/F if prefix matches with string

swapcase(!) - lowercase to uppercase and upper to lower

to_i - converts string to integer

to_sym - converts into :symbol form
